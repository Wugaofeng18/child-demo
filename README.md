# 🏰 魔法识字乐园 - 儿童识字图片生成网站

一个专为5-9岁儿童设计的中文识字学习网站，通过AI生成的主题场景图片，让学习汉字变得像玩游戏一样有趣！

## ✨ 主要特色

- 🎨 **迪士尼风格UI设计** - 色彩鲜艳、动画丰富、充满童趣
- 🤖 **AI智能生成** - 基于Nano Banana Pro模型生成高质量识字图片
- 📚 **丰富的主题场景** - 超市、医院、公园、学校等8大主题
- 💾 **本地历史记录** - 自动保存生成的图片，随时查看下载
- 📱 **完全响应式** - 支持手机、平板、电脑等各种设备
- 🔒 **隐私保护** - 所有数据仅存储在本地，保护儿童隐私

## 🚀 快速开始

### 在线使用

直接访问部署好的网站即可使用，无需安装任何软件。

### 本地运行

1. **克隆项目**
```bash
git clone [repository-url]
cd kid-literacy
```

2. **启动本地服务器**
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 或使用 PHP
php -S localhost:8000
```

3. **访问网站**
```
打开浏览器访问: http://localhost:8000
```

## 📖 使用说明

### 第一步：获取API密钥

1. 访问 [KIE AI](https://kie.ai/api-key) 注册账号
2. 获取 Nano Banana Pro API 密钥
3. 在网站中输入API密钥

### 第二步：创建识字图片

1. **选择主题** - 从8个预设主题中选择一个（如：超市、医院、公园等）
2. **输入标题** - 为你的识字图片起一个好听的名字
3. **点击生成** - AI会自动生成包含汉字拼音标注的精美图片

### 第三步：保存和分享

- **下载图片** - 将生成的图片保存到本地
- **查看历史** - 在历史记录页面查看所有生成的图片
- **批量导出** - 支持导出历史记录数据

## 🛠️ 技术栈

- **前端框架**: 纯 HTML5 + CSS3 + JavaScript
- **UI风格**: 迪士尼风格设计
- **图片生成**: Nano Banana Pro API
- **数据存储**: LocalStorage
- **字体**: Google Fonts (Comic Neue, Bubblegum Sans, Noto Sans SC)

## 🔧 配置说明

### API配置

在 `js/api.js` 中可以配置：
- API基础URL
- 模型名称
- 轮询间隔
- 超时设置

### 主题配置

在 `data/themes.json` 中可以：
- 添加新主题
- 修改现有主题词汇
- 调整主题图标和描述

### UI定制

在 `css/styles.css` 中可以：
- 修改颜色主题
- 调整动画效果
- 改变布局样式
